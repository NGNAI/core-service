# Vietnamese Response Enforcement

## Rule
Always respond in Vietnamese when providing explanations, code suggestions, or any form of assistance for this repository.

## Purpose
To ensure all interactions and documentation within this repository are conducted in Vietnamese, making it easier for Vietnamese-speaking developers to understand and contribute to the project.

## Scope
This rule applies to all code explanations, documentation, comments, and any other forms of communication within this repository.

## Implementation
- All code comments and documentation should be in Vietnamese
- All responses to questions about the code should be in Vietnamese
- All explanations of code functionality should be in Vietnamese
- All suggestions for improvements or fixes should be in Vietnamese
